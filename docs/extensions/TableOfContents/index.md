# Table Of Contents

 The Table of Contents extension allows you to add a table of contents to your editor.

## Usage

```tsx
import { TableOfContents } from 'reactjs-tiptap-editor/extension-bundle'; // [!code ++]

const extensions = [
  ...,
  // Import Extensions Here
  TableOfContents // [!code ++]
];
```
