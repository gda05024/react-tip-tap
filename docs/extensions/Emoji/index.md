# Emoji

The Document extension allows you to add a emoji to your editor.

## Usage

```tsx
import { Emoji } from 'reactjs-tiptap-editor/extension-bundle'; // [!code ++]

const extensions = [
  ...,
  // Import Extensions Here
   Emoji, // [!code ++]
];
```
