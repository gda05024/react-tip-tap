# Selection

The Selection extension allows you to add a selection to your editor.

## Usage

```tsx
import { Selection } from 'reactjs-tiptap-editor/extension-bundle'; // [!code ++]

const extensions = [
  ...,
  // Import Extensions Here
  Selection // [!code ++]
];
```
