# Trailing Node

 The Trailing Node extension allows you to add a trailing node to your editor.

## Usage

```tsx
import { TrailingNode } from 'reactjs-tiptap-editor/extension-bundle'; // [!code ++]

const extensions = [
  ...,
  // Import Extensions Here
  TrailingNode // [!code ++]
];
```
