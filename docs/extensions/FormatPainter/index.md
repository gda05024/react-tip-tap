# Format Painter

The Format Painter extension allows you to copy the format of a text and apply it to another text.

## Usage

```tsx
import { FormatPainter } from 'reactjs-tiptap-editor/extension-bundle'; // [!code ++]

const extensions = [
  ...,
  // Import Extensions Here
  FormatPainter // [!code ++]
];
```
